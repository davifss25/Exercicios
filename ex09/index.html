Cores no HTML

Cores inline

Para mudar as cores com html é só ir na tag body e usar a tag style para mudar:

  1. font-family Para mudar a fonte
  2. color Para mudar a cor da fonte

Para mudar algo especifico use a tag style dentro tag dezejada

Css interno

Para um uso melhor desse sistema mude o style dentro da tag head, mas não dentro da tag em sim, mas junto do title e os metas

Você também deve usar a tag body dentro do style e os "{}"

Css externo

Para usar o css externo use a tag "link:css" dentro do "head", abaixo do "tittle". Depois passe o mouse sobre a opção "href" e de "Ctrl + Rmb" e crie um arquivo

Lá você vai basicamente usar as mesmas tags do css interno com a adição do @charset "UTF-8"


No final das contas a melhor forma é a external
