Áudio

Para inserir um audio é só pôr a tag audio

Porém você pode pôr alguns parâmetros dentro da tag audio ao usar a tag preload

  • metadata Vai carregar apenas as informações do arquivo (temp,tam e etc)
  • none Não vai carregar nada até que o usuário decida apertar no botão de play
  • auto Vai carregar o arquivo assim que a página for carregada

    • A partir daqui essas tags não são usadas dentro do preload e sim ao lado

  • controls Vai fazer aparecer controles na tela
  • autoplay Vai fazer com que o vídeo começe a tocar automaticamente
  • loop Vai fazer o vídeo ficar repetindo após ele terminar


Abaixo da tag audio você pode usar a tag source:src para que se caso o arquivo não carrege você ponha outros arquivos ou um link de download

Vídeo

O vídeo é bem parecido com o áudio é só pôr a tag vídeo, e nela também poderemos pôr parâmetros

  • width Vai indicar a largura do vídeo
  • poster Configura a miniatura do vídeo
  • controls Indica se vão aparecer controles no player
  • autoplay Faz o vídeo dar play automaticamente

Que nem no audio você pode adicionar outros tipos de arquivos de vídeo caso o primeiro não carregue, é só usar a tag source:src

Hospedar pode ser caro

Ao hospedar um vídeo no seu site, cada pessoa que acessa-lo vai consumir tráfego, mesmo que o vídeo seja leve, então você pode usar vídeos externos como:

  • Youtube É gratuito e rápido, porém para usar-lo você tem que deixar o seu vídeo público o que é devantajozo se você quiser pôr um vídeo exclusivo para assinantes
  • Vimeo É pago, não tão rápido quanto o Youtube, porém você tem total controle de quem pode acessar o vídeo

Para adicionar esses vídeos ao seu site é só escolher a opção incorporar, copiar o código que será dado e pronto!
