Imagens dinâmicas

Para pôr imagens dinâmicas primeiro ponha a tag picture, dentro dela ponha source:media:type no media ponha o tamanho máximo da imagem g + 50px, ponha o srcset para imagem m e o tipo de arquivo no type

Depois é só por a tag img e escolher a imagem g

Agora se você quer por uma imagem dinâmica pequena fassa isso acima do sourcemediatype da imagem m e ponha o max width para o tamanho máximo da imagem m + 50px e o srcset para imagem p
