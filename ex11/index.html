Fontes

Igual as cores as fonte podem invocar emoções

Grifos, letras, caracteres

São basicamente as letras do alfabeto como: a, b, c

Família tipográfica

São basicamente a espessura da fonte

Fonte

São os simbolos que podem ser usados

Características

Varias fontes tem expressam uma sensação diferente e também uma legibilidade e leiturabilidade diferente, por exemplo a legibilidade é se é possivel ler e a leiturabilidade é se é fácil de ler

Categorias das fontes

Serifadas

Usadas antigamente para para jornais e revistas, essas fontes são muito fáceis de ler pelo fato de cada letra ter uma extenção o que faz elas "Se juntarem". Jeralmente essas fontes são usadas em textos longos, além de serem bastante usadas em títulos por chamarem atenção

Não serifadas

Invocam uma sensação de limpeza, clareza e organização

Monoespaçadas

São fontes onde cada letra ocupa o mesmo espaço, são fontes usadas para facilitar bastante a leitura e são geralmente usadas para representar códigos

Script

Tentam emular a escrita de uma pessoa e não devem ser usadas em textos grandes pois são um pouco dificeis de ler

Display

São fontes com bastante desenhos e formas diferentes, servem para causar um destaque e não devem ser usadas em textos grandes ou médios

Aplicando

Para aplicar fontes é preciso usar a tag font-family no css e depois cite algumas fonte alternativas caso a fonte primária não carregue

Tamanho das fontes

Primeiro de tudo devemos dizer que para mudar o tamanho da fonte há vários parámetros como: cm, in, pt, pc, px. Para inserir o tamalho é só usar a tag font-size

Outros estilos

Para isso é só usar a tag font-style e a tag font-weight

Simplificando tags

Para simplificar tags é só usar a tag font: e depois pôr os parâmetros que você quer italic bold 1em Arial, Helvetica, sans-serif;

Porém para que isso funcione é preciso usar cada tipo de parâmetro em ordem

  1. font-style
  2. font-variant
  3. font-weight
  4. font-size/line-height
  5. font-family

Alinhamentos

  • text-aling: left;
  • text-aling: right;
  • text-align: center;
  • text-align: justify;
