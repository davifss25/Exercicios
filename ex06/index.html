tag ol para criar a lista e tag li para cada item da lista

Para numerar cada item da lista de maneiras diferentes temos: type 1 para números, A para letras maiúsculas, a para minúsculas, I para romanos maiúsculos e i para romanos minúsculos

  1. Número
  1. Letra maiúscula
  1. Letra minúscula
  1. Romano maiúsculo
  1. Romano minúsculo

Você também pode delimitar o início da contagem com a tag start

  1. OI
