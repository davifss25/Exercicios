Cores

As cores são algo bem immportante para o seu site, pois elas geram emoções nas pessoas, por exemplo, o especialista Neil Patel afirma que na hora de comprar umm produto 90% das pessoas em 90 segundos decidem se querem ou não ele só baseado na cor

Trabalhar com a teoria das cores é uma área bem subjetiva,pois por vezes vemos marcas que adotam cores contrarias as recomendações e ainda sim fazem sucesso, então siga as recomendações, mas não se apegue muito a elas

Exemplo do que cada cor transmite

Por outro lado não basta só escolher cores bonitas, também é preciso ter uma hamonia entre elas. Isso se deve a paleta de cores que os profisionais usam para criar essa sensção e para isso vamos ver um pouco sobre o circulo cromático

Também da para dividir o circulo de várias formas

Cores complementares

São aquelas cores exatamente opostas

Cores análogas

São aquelas cores exatamente vizinhas

Cores análogas mais uma complementar

São aquelas cores exatamente vizinhas mais uma cor oposta

Cores análogas relacionadas

São aquelas cores vizinhas em que pulamos uma delas

Cores intercaladas

São cores vizinhas, mas que tem um espaço entre cada uma delas

Cores triádicas

São cores triângularmente escolhidas

Cores em quadrado

São cores quadradicamente escolhidas

Cores tetrádicas

São cores retangularmente escolhidas

Monocromacia

É quando utilizamos a mesma cor só mudando o brilho

Mudando as cores com css

Para mudar as cores de uma forma mais precisa é só ir no css e clicar no ícone da cor, assim podendo escolher a cor melhor além de poder escolher o tipo de formato de cor

E por último podemos usar gradiente ao montar as cores do nosso site, primeiro usamos a tag background-image depois linear-gradient para criar um gradiente no seu site ou radial gradient para criar um gradiente circular e por ultimo as tags deg para controlar a intensidade do degrade e as cores que você deseja mesclar

style body{ background-image: linear-gradient(90deg, yellow, red); }
